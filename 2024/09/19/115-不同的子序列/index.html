115.不同的子序列

Posted on 2024-09-19 Edited on 2024-09-23

题目描述：
给你两个字符串 s 和 t ，统计并返回在 s 的 子序列 中 t 出现的个数，结果需要对 10^9 + 7 取模。
示例 1：
输入：s = “rabbbit”, t = “rabbit”
输出：3

示例 2：
输入：s = “babgbag”, t = “bag”
输出：5

提示：
1 <= s.length, t.length <= 1000
s 和 t 由英文字母组成

解题法：

  1. 回溯法。
    首先找到t的第一个字符匹配的第一个地方。，然后计算后面字符的匹配的count。然后计算t的第一个字符匹配的第二个地方，计算后面字符匹配的count… 但是时间复杂度很高，具体复杂度大概为(s.size()/26)^n。

正确做法是使用动态规划的做法。定义一个二维数组dp[i][j]表示s[0,i-1]与t[0,j-1]的匹配数量
递推公式
如果 s[i-1] == t[j-1]，那么 dp[i][j] = dp[i-1][j-1] + dp[i-1][j]，即当前字符匹配时，可以选择匹配它或者跳过它。
如果 s[i-1] != t[j-1]，那么 dp[i][j] = dp[i-1][j]，即跳过 s[i-1]，继续匹配。
初始值：找到第一个字符匹配的位置k，那么之前的所有dp[][]均为0，dp[k+1][k+1]为1，之后就进行动态规划即可。

注意：
对于二维动态规划，dp[i][j]均要使用两次for循环。
对于初始值的选择，要考虑特殊情况。
比如为空、边界等。
对于i j循环哪一个是外层循环，哪一个是内层循环，主要根据递推公式来进行考虑。
对于上面这个例子， 递推公式是

  1. 最长回文子序列
    给你一个字符串 s ，找出其中最长的回文子序列，并返回该序列的长度。
    子序列定义为：不改变剩余字符顺序的情况下，删除某些字符或者不删除任何字符形成的一个序列。
    思路：
  2. 状态定义：dp[i][j]表示子串s[i…j]的最长回文子序列的长度。
  3. 状态转移方程：
    • 如果s[i] == s[j]，那么 dp[i][j] = dp[i+1][j-1] + 2。
    • 如果 s[i] != s[j]，那么 dp[i][j] = max(dp[i+1][j], dp[i][j-1])。
  4. 初始化条件
    • 当i==j时，表示单个字符，dp[i][j] = 1;
  5. 最终条件答案保存在dp[0][n-1]中，其中n是字符串的长度。 注意
  • 尤其注意递推公式的划分。对于s[i] == s[j]，那么 dp[i][j] = dp[i+1][j-1] + 2，情况显而易见。对于s[i] != s[j]，那么 dp[i][j] = max(dp[i+1][j], dp[i][j-1])，要懂得进行分类。
  • 尤其注意动态规划的解法本质上就是先计算子问题的解并记录下来，然后再根据子问题的解和递推公式求解新问题。所以循环的设定应该是问题由小到大。对于上面不同的子序列。其递推公式是dp[i][j] = dp[i-1][j-1] + dp[i-1][j]和 dp[i][j] = dp[i-1][j]，为了求得dp[i][j]的值，需要先求得dp[i-1][j-1]和dp[i-1][j]的值，所以i = 1m, j = 1n的循环完全可以实现先求解子问题。

714. 买卖股票的最佳时机含手续费

给定一个整数数组 prices，其中 prices[i]表示第 i 天的股票价格 ；整数 fee 代表了交易股票的手续费用。
你可以无限次地完成交易，但是你每笔交易都需要付手续费。如果你已经购买了一个股票，在卖出它之前你就不能再继续购买股票了。
返回获得利润的最大值。
注意：这里的一笔交易指买入持有并卖出股票的整个过程，每笔交易你只需要为支付一次手续费。

使用动态规划。
注：dp数组的主要目的是为了记录状态，所以定义一个能够记录状态同时又很简洁的dp数组是一个很重要的事情。对于这道题，可以使用两个一维数组。dp_hold[n]和dp_unhold[n]，前者记录当第i天的时候持有股票时的收益，后者记录第i天不持有股票的收益。

  • 初始条件dp_hold[0] = -prices[0]; dp_unhold[0] = 0;
  • 递推公式dp_hold[i] = max(dp_unhold[i-1] - prices[i],dp_hold[i-1]); dp_unhold[i] = max(dp_hold[i-1] + prices[i] - fee, dp_unhold[i-1]);
  • 结果：dp_unhold[n-1];

813. 最大平均值和的分组

给定数组 nums 和一个整数 k 。我们将给定的数组 nums 分成 最多 k 个非空子数组，且数组内部是连续的 。 分数 由每个子数组内的平均值的总和构成。
注意我们必须使用 nums 数组中的每一个数进行分组，并且分数不一定需要是整数。
返回我们所能得到的最大 分数 是多少。答案误差在 10-6 内被视为是正确的。

  1. 使用动态规划。定义一个二维动态数组dp[i][j]表示前i个数分成j分的得分。
  2. 递推公式：dp[i][j]=max(dp[i][j],dp[t][j−1]+avg(nums[t+1:i]))，其中，t表示j-1到i-1
  3. 初始条件dp[i][1] = 前i个数的平均 注
  4. 一般的做法是，dp数组的0索引视为无意义。同时也要注意相关的数组的索引设置。比如本题中，dp[i][j]表示前i个数分成j组，那么dp定义时dp(n + 1, vector(k + 1, 0.0));
  5. 同时prefixSum表示前n个数的和，因此定义的时候为prefixSum(n + 1, 0.0);
